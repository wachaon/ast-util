# ast-util
**ast util** は **@bable/parser** で解析した ast を弄る為のモジュールを集めたものになります。

| 用途                     | 名前       | 説明                 |
|:-------------------------|:-----------|:---------------------|
| パーサー                 | `parser`   | **@babel/parser**    |
| トラバーサー             | `visit`    |                      |
| ジェネレーター           | `generate` | **@babel/generator** |
| 余分なプロパティーの削除 | `removal`  |                      |
| 表示                     | `inspect`  |                      |
